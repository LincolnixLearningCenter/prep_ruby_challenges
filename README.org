** Power
   
   Write a method /power/ which takes two integers (/base/ and /exponent/) and 
   returns the /base/ raised to the power of /exponent/. Do not use Ruby's /**/
   operator for this!

   #+begin_src ruby
     > power(3,4)
     => 81 # (3*3*3*3)
   #+end_src
